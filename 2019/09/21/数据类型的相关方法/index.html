数据类型以及相互之间的转化

数据类型

基本数据类型

1. 数字（number）
    * NaN是数字类型，它表示不是一个有效数字，它和谁都不相等（包括自己）
2. 字符串（string）
3. 布尔（boolean）
4. 空（null）
    typeof null  //"object"
5. 未定义（undefined）
    null==undefined  //true

引用数据类型

对象类型object

1. 普通对象｛｝
2. 数组对象{}
3. 正则对象
4. 日期对象
5. 数学函数对象
6. 类数组对象
7. 实例对象
8. 原型对象

函数function

函数也是对象。它有三种角色：普通函数、构造函数、对象

特殊数据类型

  1. symbol()
    symbol用于创建一个唯一值

数据类型之间的相互转化

其他数据类型转化为number类型

  1. isNaN():检测一个数是否为非有效数字类型，返回值是一个boolean类型
    检测机制：当检测的是一个引用类型时，先调用String()转换为字符串，然后再调用Number()方法转换为数字，最后检测是否为非有效数字

  2. Number():讲一个数转换为数字类型
    转换机制：基础数据类型直接转换，引用数据类型先调用String()方法转换为字符串，在调用Number()转换为数字类型

  3. 基于parseInt()和parseFloat()转换为数字类型
    转换机制：从左到右依次查找，只要遇到一个非有效数字，结果就是NaN
    parseInt():将一个有效数字转换为整数
    parseInt(n,m):支持传递两个参数，表示把m进制中的n转换为十进制下的数

    parseFloat():将一个有效数字转换为一个小数

  4. 在if条件判断中也会进行转换
    对象VS数字：把对象转换为数字
    对象VS布尔：把对象转换为数字，把布尔也转换为数字
    对象VS字符串：把对象转换为数字，把字符串也转换为数字
    字符串VS数字：字符串转换为数字
    字符串VS布尔：都转换为数字
    布尔VS数字：把布尔转换为数字

其他类型转换为字符串

  1. 基于alert/confirm/prompt/document.write等方法输出内容的时候，会把输出的值转换为字符串，然后再输出
  2. 基于“+”进行字符串拼接的时候
  3. 把引用类型值转换为数字的时候，首先会转换为字符串，然后再转 换为数字
  4. 给对象设置属性名，如果不是字符串，首先转换为字符串，然后再 当做属性存储到对象中（对象的属性只能是数字或者字符串）
  5. 手动调用toString/toFixed/join/String等方法的时候，也是为了转换为 字符串

    其他值转换为布尔类型

  6. 基于!/!!/Boolean等方法转换
    转换规则：只有0/NaN/‘’/null/undefined五个值转换为布尔的false,其余都是转换为 true

    引用类型的类型转换

    特殊情况：“==”在进行比较的时候，如果左右两边的数据类型不一样，则先转换为相同的类型，再进行比较
  • 引用类型要注意，比较的是地址是否相同
